## Quick Start

1. Clone the repo
   ```bash
git clone https://github.com/COE-Free/GAIA-Triad-Cycle1-2026.git
   cd GAIA-Triad-Cycle1-2026
   ```
2. Run the digital twin (once populated)
   ```bash
   docker compose up
   # or python buckeye_sanctuary_master.py
   ```
